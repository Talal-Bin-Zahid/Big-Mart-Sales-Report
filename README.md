## üìå Project Overview

The **Big Mart Sales Analysis Dashboard** is a Power BI project that provides a comprehensive view of sales performance across different product categories, outlet types, store locations, and pricing structures.

This report helps retail decision-makers and business stakeholders identify **top-performing product segments, profitable outlet characteristics, and pricing impacts** on sales, ultimately enabling smarter inventory and expansion strategies.

| -------------------------- | ----------------- | ------------------------------------------------- |
| **Total Sales**            | **18.59 Million** | Total revenue generated from all Big Mart outlets |
| **Total Outlets**          | **10**            | Total number of store outlets in the dataset      |
